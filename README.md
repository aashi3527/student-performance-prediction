# Student Performance Prediction

A machine learning project to predict student exam scores based on behavioural and demographic features.

## Tech Stack
Python, Pandas, NumPy, Scikit-Learn, Matplotlib

## Models Used
Linear Regression, Decesion Tree, Random Forest

## Results
- Best model: Linear Regression
- R² Score: ~0.79

## Dataset
1,000+ student records with features including study hours, attendance rate, and parental education level.
